# 선택자 목록

  

같은 속성이 있을 때 묶어서 사용

,를 이용해서 그룹으로 묶어줌

```JavaScript
.book-description {
  font-size: 12px;
  font-weight: 400;
  line-height: 17px;
}

.book-info {
  background-color: #000000;
  border-radius: 12px;
  color: \#ffffff;
  font-size: 12px;
  font-weight: 400;
  line-height: 17px;
  margin: 24px 0;
  padding: 12px 32px;
  text-align: center;
}
```

```JavaScript
.book-description, .book-info {
  font-size: 12px;
  font-weight: 400;
  line-height: 17px;
}

.book-info {
  background-color: #000000;
  border-radius: 12px;
  color: \#ffffff;
  margin: 24px 0;
  padding: 12px 32px;
  text-align: center;
}
```
