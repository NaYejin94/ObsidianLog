
아이돌 정보를 입력하면 POST요청으로 아이돌을 추가한다

✅ 작업 내용

- 아이돌 정보 입력 폼 추가
    
    이미지 버튼 ( + )를 누르면 업로드되는 이미지 보이게 구현
    
    추가하기 버튼 누르면 확인 창 추가
    
    뒤로가기 버튼을 누르면 전 페이지로 이동 (어느 페이지에서 추가할건지 의논 필요)
    
    디자인 추후 적용
    
    확인창에 입력한 아이돌 정보 넘겨주기
    
    인풋에 입력하지 않은 값이 있다면 확인 버튼 disable 시키고 필수 입력 알리는 div 추가
