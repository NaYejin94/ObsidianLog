# Image 컴포넌트 소개

  

- Next.js에서는 이미지를 사용할 때 Next.js 서버를 한 번 거쳐서 이미지 최적화를 한 다음 사용할 수 있도록 해 줌
- `width`와 `height` 값을 지정하거나 또는 `fill`이라는 Prop을 사용

  

- width, height 사용

```TypeScript
 <Image
        src="/images/product.jpeg"
        width={300}
        height={300}
        alt="상품 이미지"
      />
```

  

- fill 사용

```TypeScript
<div
	style={{
	  position: 'relative',
	  width: '100%',
	  height: '300px',
	}}
>
	<Image
	  src="/images/product.jpeg"
	  fill
	  alt="상품 이미지"
	  style={{
	    objectFit: 'cover',
	  }}
	/>
</div>
```

# 빌드하기와 실행하기

  

- **빌드 (Build)**
    
    : 리액트 코드를 브라우저가 실행할 수 있게 변환
    
    : 서버 실행에 필요한 코드들을 생성
    
    : `npm run build`
    

  

- `.next` 파일
    
    : Next.js가 서버에서 실행할 파일
