# 테이블 스타일링

테두리 만들기

```JavaScript
table{
    border: 1px solid red;
    border-collapse: collapse; -> 테두리 공간 없애기
    border-spacing : 15px -> 테두리 공간 늘리기
}

td,th{
    border:  1px solid blue;
}

thead th, thead td {
    border: 1px solid green
}
```
