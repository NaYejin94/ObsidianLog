# 순서 리스트

  

순서 리스트 태그 → <ol>태그 안에 <li>태그

(ol = ordered list)

```JavaScript
<ol>
        <li>라라랜드 </li>
	    <li> 명량</li>
	    <li> 극한직업</li>
        <li> 신과함께: 죄와 벌</li>
        <li> 국제시장</li>
        <li>어벤져스: 엔드게임
    </ol>
```

  

  

# 순서 없는 리스트

  

<ul>태그 안에 <li>태그

(unordered list)

```JavaScript
<ul>
  <li> 한국 영화</li>
   <li>  외국 영화</li>
  <li>  드라마</li>
  <li>  예능</li>
  <li>  영화</li>
  <li>  프로야구</li>
</ul>
```

  

  

# 리스트 스타일링

  

: <ol type=””> or <ul type=””>

: css로 표현하기

```JavaScript
<style>
	ol {
			list-style: disc;
	}
	</style>
```
